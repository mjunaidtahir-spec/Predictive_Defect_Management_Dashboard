# AI-Driven Predictive Defect Management Dashboard

## Project Overview

This project demonstrates an AI-powered defect management analytics solution designed to help project teams, PMOs, QA teams, and delivery managers proactively identify high-risk defects and predict potential escalations.

The solution combines:

- Data Analytics
- Machine Learning
- Risk Scoring
- Predictive Modelling
- Interactive Dashboard Visualization

The objective is to move defect management from reactive reporting toward proactive decision-making.

---

# Business Problem

Traditional defect dashboards mainly show:

- Number of defects
- Defect status
- Priority distribution
- Aging reports

However, project leadership needs answers to questions such as:

- Which defects are most likely to become critical?
- Which defects require immediate management attention?
- Which factors contribute to escalation risk?
- Where should testing and delivery teams focus their effort?

This project addresses these questions using predictive analytics.

---

# Solution Architecture

The solution consists of three major components:

## 1. Data Exploration & Preparation

Notebook:

`01_Data_Exploration.ipynb`

Activities:

- Data profiling
- Data quality checks
- Feature analysis
- Defect pattern identification

---

## 2. AI Risk Scoring Engine

Notebook:

`02_Risk_Scoring_Engine.ipynb`

Capabilities:

- Calculates defect risk score
- Identifies risk levels
- Determines risk drivers

Risk factors include:

- Priority
- Customer impact
- Defect age
- Escalation indicators
- Discussion activity

---

## 3. Predictive Escalation Model

Notebook:

`03_Escalation_Prediction.ipynb`

Machine Learning approach:

- Feature engineering
- Model training
- Prediction generation
- Model evaluation

Model output:

- Predicted escalation probability
- Predicted escalation status
